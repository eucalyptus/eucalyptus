Console Login

This screen allows you to log in to the Eucalyptus console. If you've forgotten your password or don't have login credentials, please contact your system administrator.

Login to the Console

  1. Type your account name into the Account name text box. If your system administrator embedded the account information in the console URL, this text box will not appear.
  2. Type your user name into the User name text box.
  3. Type your password into the Password text box.
  4. Click the Login button.
