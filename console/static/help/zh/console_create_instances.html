创建虚拟机实例

你可以在本窗口创建虚拟机实例.

选择一个虚拟机模版

在这里选择一个虚拟机模板来创建虚拟机.

  1. 选择用过滤下列列表或者搜索文本框来定位你要选择的虚拟机模板.
  2. 在选择模板列表中选中你要的模板.
  3. 点击 下一步: 选择虚拟机类型 按钮，进入 虚拟机类型 版面.

选择虚拟机类型

在本版面你可以指定虚拟机的大小，虚拟机数目，可用区域

  1. 通过点击相应的图标指定虚拟机大小.
    提示: 通过点击虚拟机大小连接可以得到虚拟机大小的详细信息.
  2. 可选则的，你可以在 虚拟机数目 文本框，输入你要创建的虚拟机的数目
  3. 如果不用缺省区域，你可以在 可用区域 下拉列表框中指定一个.
  4. 点击 下一步: 选择安全管理 按钮 进入 安全管理 版面.

安全管理

在本版面，你可以为要创建的虚拟机实列指定密钥对和安全组. 提示: 你在这里创建的密钥对和安全组会自动被虚拟机实例使用.

  1. 在密钥对 下拉列表.选择一个密钥对
    提示: 点击 创建密钥对 link to display the Create new key pair 对话框，你可以创建新的密钥对.
  2. 在 安全组下拉列表框， 选择一个安全组.
    提示: 点击 新建安全组 连接，可以显示 新建安全组 对话框. 你可以在对话框里新建一个安全组
    提示: 如果你选择缺省安全组，请加入相关规则到缺省安全组来保证你一访问你建立的虚拟机实例.
  3. 点击 选择高级选项 连接. 你可以设置一些高级选项
  4. 点击 发布虚拟机实例 按钮.

指定高级选项

在本版面，你可以为虚拟机指定一些高级选贤，包括制定用户数据，指定内核和内存盘的标示，私有网络和扩展存储.

  1. 通过在 用户数据 文本框输入或者在 附件 连接上传文件. 你可以指定用户自定义的数据
  2. 在 内核ID 下拉列表中.指定一个内核
  3. 在 内存盘ID 下拉列表中，指定一个内存盘.
  4. 点击 网路 选项框，为虚拟机实例指定私有地址. 私有地址不能直接连到Internet，要连接到Internet,需要通过NAT或者为虚拟机设置弹性IP.
  5. 你可以为虚拟机添加一个或者多个扩展存储设备:
    1. 在 存储卷 下拉列表中指定存储设备类型(EBS或者ephemeral).
    2. 在 映射 文本框输入设备映射名(例如, sdc 对应 /dev/sdc).
    3. 如果你想从存储快照建立一个存储卷，可以在 the snapshot using the 从快照建立 下拉列表中选择一个快照来创建
    4. 在 大小 (GB) 文本框，输入存储设备的大小.单位为G
    5. 选择 在终止后删除 选项 如果你想删除这个存储设备当虚拟机被终止后.
    6. 点击 继续添加 按钮 继续为虚拟机添加存储设备
  6. 点击 发布虚拟机实例 按钮，发布虚拟实例.
