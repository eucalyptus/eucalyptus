管理安全组规则

在本对话框,你可以编辑安全组规则.

新添安全组规则

  1. 在 协议 下拉列表选择一种协议.
  2. 在 端口范围 文本框输入端口范围.
  3. 选择进入流量控制类型:

    要用IP地址段控制, 选择 IP 地址 单选按钮并且在文本框输入CIDR范围.

    提示: 更多关于 CIDR 的解释, 请参考 CIDR notation Wikipedia 文章.

    要用安全组控制, 请选择 其它安全组 单选按钮并且在文本框输入安全组名字.

    提示: 使用格式 "userid/groupname"来指定其它用户帐号的安全组.
  4. 点击 TBD 按钮.

删除安全组规则

点击现存安全组规则旁边的删除 连接.可以直接删除该安全组规则
