创建安全组

Eucalyptus 使用安全组来控制对云的网络访问.一个安全组包含一组规则，在指定的网络协议和端口，控制进入虚拟机的网络数据包.

在控制台创建安全组

  1. 在名字 文本框输入安全组的名字.
  2. 在 描述 文本框输入安全组的描述信息.
  3. 你可以选择性的创建一条或者多条规则, 每条规则授权对一个或者一组端口和协议的入栈访问.
    1. 在 协议 下拉列表中选择协议.
    2. 在 端口范围 文本框输入端口段.
    3. 选择进入流量控制类型:

      如果想通过源IP地址设置控制, 选择 IP 地址 单选按钮,在文本框输入CIDR段.

      提示: 有关 CIDR 的解释, 请参考 CIDR notation Wikipedia 文章.

      如果想通过安全组设置控制, 选择 其它安全组 单选按钮，在文本框输入安全组名字.

      提示: 使用"userid/groupname"格式，可以指定其它用户的安全组.
    4. 点击 创建其它规则 按钮建立更多规则.
  4. 点击 创建安全组 按钮.
