发布更多的虚拟机实例

在本对话框,你可以以现有的虚拟机实例为蓝本,创建一个或者多个具有相同特征的新虚拟机实例.

指定虚拟机实例数目

在本版面输入要创建的虚拟机实例数目.

  1. 在 虚拟机实例数 文本框输入虚拟机实例数目.
  2. 要设置高级选项,点击 选择高级选项 连接.
  3. 点击 发布虚拟机实例 按钮,创建新的虚拟机实例.

指定高级选项

在版面你可以为虚拟机实例指定一些高级选项. 你可以添加用户数据, 指定内核ID和内存盘ID, 指定私有网络, 和附加磁盘.

  1. 要定制用户数据,你可以在 用户数据 文本框输入数据或者在通过 上传文件 连接上传文件.
  2. 要为虚拟机映像指定内核,请在 内核ID 下拉列表中选择.
  3. 要为虚拟机映像指定内存盘,请在 内存盘ID 下拉列表中选择.
  4. 点击 网络 选项框, 你可以为虚拟机实例指定私有IP. 私有地址不能直接连到Internet，要连接到Internet,需要通过NAT或者为虚拟机设置弹性IP.
  5. 你可以为虚拟机添加一个或者多个扩展存储设备:
    1. 在 存储卷 下拉列表中指定存储设备类型(EBS或者ephemeral).
    2. Type the desired device mapping into the Map text box (for example, sdc for /dev/sdc). 在 映射 文本框输入设备映射名(例如, sdc 对应 /dev/sdc).
    3. 如果你想从存储快照建立一个存储卷，可以在 the snapshot using the 从快照建立 下拉列表中选择一个快照来创建
    4. 在 大小 (GB) 文本框，输入存储设备的大小.单位为G
    5. 选择 在终止后删除 选项 如果你想删除这个存储设备当虚拟机被终止后.
    6. 点击 继续添加 按钮 继续为虚拟机添加存储设备
  6. 点击 发布虚拟机实例 按钮，发布虚拟实例.
